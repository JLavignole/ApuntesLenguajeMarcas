Apuntes de GitHub y sus Comandos en el CMD

  • Crear un Repositorio en GitHub

    1. Crear una cuenta en GitHub

    2. Crear un nuevo Repositorio:
      • Haz clic en el botón "New" o "Nuevo" en tu página de inicio de GitHub.
      • Asigna un nombre a tu repositorio
      • Decide si el repositorio será público o privado.
      • Puedes agregar un archivo README para describir tu proyecto.
      • Haz clic en "Create repository".

    Comandos básicos de Git en CMD

    1. Configuración inicial:
      • Para configurar el nombre, ponemos: git config --global user.name
      • Para configurar el correo, ponemos: git config --global user.email

    2. Inicializar un repositorio
      • Para inicializar un repositorio en la máquina local, usamos: git init

    3. Clonar un repositorio
      • Para clonar un repositorio existente en GitHub a tu máquina local, usamos: git clone https://github.com/usuario/nombre-del-repositorio.git

    4. Comandos para gestionar cambios:
      • Para añadir archivos en el área de preparacion, ponemos: git add archivo.txt para añadir un archivo en concreto
      • Para añadir archivos en el área de preparacion, ponemos: git add . para añadir todos los archivos
      • Para realizar un comit, ponemos: git commit -m "Mensaje descriptivo del commit"
      • Para enviar al los cambios al repositorio remoto, ponemos: git push origin main suponiendo que tu rama principal se llame main
      • Para crear una rama nueva, ponemos: git branch nombre-de-rama
      • Para descargar los cambios recientes del repositorio remoto, ponemos: git pull
      • Para clonar repositorio de la nube al local utilizamos: git clone URL del repositorio
